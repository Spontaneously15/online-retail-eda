# Online Retail Exploratory Data Analysis (EDA) with Python

This project analyzes the Online Retail dataset (2010–2011) to uncover sales trends, customer purchasing behavior, and top-selling products.  
It demonstrates real-world data cleaning, visualization, and insight generation using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analyses
- Data cleaning and handling missing values
- Exploratory statistics
- Top-selling products
- Revenue analysis
- Monthly and weekly sales trends
- Customer behavior
- Outlier detection

## Notebook
The complete analysis is available in the Jupyter Notebook:  
**online_retail.ipynb**

## Sample Insights
- Total Transactions: **397,884**
- Unique Customers: **4,338**
- Total Revenue: **£8.9M+**
- November is the busiest month
